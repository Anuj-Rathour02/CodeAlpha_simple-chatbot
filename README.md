# CodeAlpha_simple-chatbot
A simple rule-based chatbot built using Python. The chatbot interacts with users through predefined commands and provides appropriate responses. This project demonstrates fundamental Python concepts such as functions, loops, conditional statements, string handling, and user input/output.  Features

## Features
•User Greeting System
•Personalized Name Recognition
•Current Time Display
•Joke Response
•Help Menu
•Conversation Counter
•Exit Functionality
•Rule-Based Responses

## Commands Supported
•hello
•how are you
•your name
•my name
•time
•tell me a joke
•thank you
•help
•bye

## Technologies Used
•Python

## Concepts Used
•Functions
•While Loops
•If-Else Statements
•User Input and Output
•String Manipulation
•Time Module
